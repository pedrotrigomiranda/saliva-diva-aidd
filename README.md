# Saliva Diva Website

This is the website for Saliva Diva, built with Next.js and Material UI.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- Next.js
- Material UI
- TypeScript

## Project Structure

- `app/` - Next.js app directory
- `public/` - Static assets
- `components/` - Reusable components (to be added)

## Deployment

Deploy on Vercel or any other platform supporting Next.js.
